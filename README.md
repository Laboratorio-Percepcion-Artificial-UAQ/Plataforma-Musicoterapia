# Plataforma digital educativa sobre musicoterapia perioperatoria

Segunda versión local de una landing page educativa médica desarrollada con HTML, CSS y JavaScript.

## Cambios principales de la versión 2

- Lenguaje interdisciplinario: salud, personal sanitario y equipo de salud.
- Sustento académico ampliado en definición, beneficios, mecanismos e instrumentos.
- Sección robusta de evidencia científica con revisiones sistemáticas, metaanálisis y ensayos clínicos.
- Apartado de instrumentos validados: APAIS y STAI-S.
- Cuestionario interactivo mantenido como herramienta educativa no diagnóstica.
- Recomendaciones prácticas para pacientes.
- Guía breve para personal de salud.
- Ilustraciones contextuales del entorno perioperatorio.

## Visualización local

Abra `index.html` directamente o use Visual Studio Code con Live Server.
